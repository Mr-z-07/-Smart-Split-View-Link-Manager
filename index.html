Privacy Policy for Smart Split View Link Manager

Effective Date: April 25, 2026

This extension collects no personal data. All links and lock settings are stored locally on your device using Chrome's storage.local API. No information is sent to any server.

You can delete all data at any time via the extension's Delete or Reset All buttons, or by uninstalling the extension.

For questions, contact the developer via the GitHub repository where this policy is hosted.
