# Privacy Policy for Smart Split View Link Manager

**Effective Date:** April 25, 2026  

This extension ("Smart Split View Link Manager") respects your privacy. This policy describes what data the extension collects, how it is used, and your rights.

## Data Collection & Storage
- **No personal information is collected** – the extension does not send any data to external servers.
- All data (your saved links and their lock/unlock status) is stored **locally** on your device using Chrome’s `storage.local` API.
- No browsing history, IP addresses, or tracking identifiers are ever accessed or transmitted.

## How Your Data Is Used
- Saved links are only used to open popup windows or tabs as you request via the extension’s buttons.
- Lock/unlock status is stored solely to remember your preferences between browser sessions.

## Third‑Party Access
- The extension does not use any external analytics, ads, or third‑party services.
- The SortableJS library (loaded from a CDN) runs only within the extension’s popup and does not collect data.

## Data Deletion
- You can delete individual links using the **🗑️ Delete** button.
- The **Reset All** button removes all saved links and restores the default set.
- To completely remove all extension data, uninstall the extension from Chrome.

## Changes to This Policy
Any future changes will be posted on this page. Continued use of the extension after changes constitutes acceptance.

## Contact
If you have questions, please open an issue on the extension’s GitHub repository (the link you obtained this policy from).

---

*This extension operates entirely offline and locally – no data ever leaves your browser.*
